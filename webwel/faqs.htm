 WELCOM requires STATA version 11 or higher.

How do I add the WELCOM menu to Stata?
You have three options to add the command “WELCOM” to your Stata menu:  

1) Add the command “_welcom_menu” to your Stata profile file “c:/ado/personal/profile.do”. This will ensure that Stata loads WELCOM each time is launched.  

2) If you don’t have a “profile.do” file, you can create one by saving the command “_welcom_menu” in a do file with the name “profile.do” in the directory “c:/ado/personal/”.  

3) Alternatively, execute the command “_welcom_menu” when you launch Stata and every time you use the WELCOM menu. 

What data do I need?

WELCOM requires data from household consumption or expenditure surveys that contain information on expenditure of the goods expected to be affected by the change in competition.
