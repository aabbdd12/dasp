DASP 2.3 improved modules.


This page contains some the improved modules of DASP 2.3, and this, based on a set of requests of users across the world. We report in the following list the name of the module, the description of the improvement and the link to download the updated module as desired.



clorenz.ado


 Instead of drawling the Lorenz curve based on only 100 percentiles, the user can indicate the number of these percentiles with the PERCentiles()
ifgt_mi.ado







 During the last years, there are an upsurge interest to treat as much as possible the missing values with the multivariate imputations technic (mi). Even if the general rule for the practitioner was to drop the observations with missing values, the mi technic can improve the accuracy of estimated distributive indices in the case where the number of missing values is higher and where we can treated this with the mi technic. the module ifgt_im enables to estimate the FGT poverty indices with the mi data
cdomc_mi.ado


 The improvement is similar to that of the ifgt_mi, but it concernecs here the consumption dominance curve.
