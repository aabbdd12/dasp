The GEPWE Stata module
 
The GEPWE Stata module can be used to generate the variable percentile weight regression, as developped by Araar (2016). For more details, see the following technical note and the example.do file. 
 
Reference:
Araar Abdelkrim (2016), Percentile Weighed Regression , PEP-PMMA technical note series 2016-01.
