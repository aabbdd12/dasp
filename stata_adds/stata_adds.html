Stata modules
By Araar Abdelkrim
Here below we propose a set of Stata routines which can be used for the study of the distributive analysis, as well as, some specific econometric topics. For each of the proposed Stata modules, a short description is provided, as well as, the available *.ado *.hlp and *.dlg files. In addition, a zipped format is provided to load all the files and the material of each covered topic.
