FAQs

What Stata version is required?

 DASP 3.0 requires STATA version 15.0 or higher.

DASP  2.3 requires STATA version 10.0 or higher.

How to install or update DASP

  Method 1: installing DASP modules in PLUS directory (recommended).

  1. After downloading the DASP package and dezipping it, copy the folder dasp in the directory c/temp:
  2. Make sure that you have
  3. c:/temp/dasp3/dasp1.pkg,
  4. c:/temp/dasp3/dasp2.pkg,
  5. c:/temp/dasp3/dasp3.pkg,
  6. c:/temp/dasp3/dasp4.pkg,
  7. c:/temp/dasp3/dasp5.pkg,
  8. c:/temp/dasp3/dasp6.pkg and
  9. c:/temp/dasp3/stata.toc
  10. In the Stata command windows, type the syntax

     net from c:/temp/dasp3

 Type the commands

net install dasp_1.pkg, force

net install dasp_2.pkg, force

net install dasp_3.pkg, force

net install dasp_4.pkg, force

net install dasp_5.pkg, force

net get     dasp_6.pkg, force
