DASP 2.3 improved modules.
This page contains some of the  improved modules of DASP 2.3, and this, based on the requests of DASP users. We report in the following list the name and the description of the improvement.


clorenz.ado


 Instead of drawling the Lorenz curve with only 100 percentiles, the user can indicate the number of percentiles with the option PERCentiles()
ifgt_mi.ado







 During the last years, there is an upsurge interest to treat the missing values with the multivariate imputations technic (mi). Even if the general rule for practitioners was to drop the observations with missing values, the mi technic can improve significantly the estimated indices, especially, where the number of missing values is high, and where it is appropriate to treat the bias whith the mi technic. the module ifgt_mi enables to estimate the FGT poverty indices with the mi for of data.

cdomc_mi.ado


 The improvement is similar to that of the ifgt_mi, but it concerns the estimation of the consumption dominance curve.
